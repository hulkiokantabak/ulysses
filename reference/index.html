The reference documents

Three base documents, built while the book was fresh. They are the deliverables' source of truth, written to outlive any particular design. Each cites by episode rather than page, so it survives an edition change; two carry interactive versions.

  • The Character Map — 51 characters in four circles, with the interactive ring.
  • The World, the City, and the Day — Dublin and the world, 1904.
  • The Itinerary — place and hour, episode by episode, with the map and time rail.

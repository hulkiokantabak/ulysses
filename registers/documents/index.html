Every document

The PDF register: durable, portable, offline. Everything on the site that exists as a document, in one place.

  • Ulysses Reading Companion — the full Q&A of the read (117 pages) — context
    Hulki Okan Tabak — the primary record of a first reading of Ulysses, 7–17 July 2026. The reader's questions are reproduced as spoken, dictation slips included (anchor A6).
  • Ulysses Reading Questions — questions only (17 pages) — context
    Hulki Okan Tabak — the questions of the reading, separated from their answers; the one artifact sent to the independent second build (anchor A17).
  • Four Odysseys: Homer, Wilson, Nolan, Joyce — a comparative handoff — context
    Prepared 18 July 2026 from day-one reviews of Christopher Nolan's The Odyssey (2026); every source is cited inside the document. Its confidence markings — High / Medium / Partial / Unconfirmed — are the document's own and survive into anything published (anchor A5). This is dimension 16's SOURCE MATERIAL, not its PDF register: it is a linear document made outside this project, with no internal navigation, no cross-references and no outbound links, so it does not meet base document 00's definition of the register (catch C-26). It is filed under Reading, like the other two source documents.
  • The Difficulty Curve — the document (hyperlinked PDF, 9 pages) — context
    The first PDF built by this project's own pipeline, 19 July 2026 — hyperlinked and cross-linked as base document 00 requires: contents to section to back, cross-references opening the 117-page reading record at the cited page, and a bookmark outline. Built from the dimension's HTML register, so the two carry the same material; n = 1 in the document's own voice (anchor A10), and the caveats are part of the document. Set in DejaVu, embedded under the Bitstream Vera licence (RIGHTS.md clause 6).
  • The Women's Dublin — the document (hyperlinked PDF) — context
    Built by this project's pipeline, 19 July 2026 — hyperlinked and cross-linked per base document 00: contents to section to back, and an outbound link to the public-domain 1922 text it quotes throughout. Sixteen women, 181 appearances, every one collected against the 1922 text and adversarially verified; the corrections are disclosed in the document's own caveats. How each woman enters the text — on stage, as a letter, a body part, or men's talk — is recorded per anchors A3 and A5. Episode titles are scaffolding (A2).
  • The Governed City — the document (hyperlinked PDF) — context
    Built by this project's pipeline, 19 July 2026 — hyperlinked and cross-linked per base document 00. Thirteen instruments of rule across the eighteen episodes, collected against the 1922 text and adversarially verified. The matrix marks presence, not weight — one constable and a whole cavalcade are each one mark — and the document says so in its own voice. Episode titles are scaffolding (A2).
  • The Constraint Model — the document (hyperlinked PDF) — context
    Built by this project's pipeline, 19 July 2026 — hyperlinked and cross-linked per base document 00. The model is INTERACTIVE and a PDF cannot be operated: this document is the model's record, not its substitute, and it says so in four places while pointing at the operable version. All sixteen modelling assumptions are listed, with the source's own 16-versus-17 count discrepancy shown rather than resolved (anchor A4). A toy model in the economist's sense, in its own voice.
  • Four Odysseys — the document (hyperlinked PDF) — context
    Built by this project's pipeline, 19 July 2026, and the register dimension 16 was owed — the handoff document under Reading is its source material, not this. Its Wilson claims carry the markings from the public-source verification of the same day (C-30): the equal line count High, the figure 12,110 withdrawn as untraceable, 'strict' withdrawn in favour of Wilson's own word 'regular', and the 'slave' choice High and primary with the Fagles contrast verified against printed text. Every claim about Nolan's film is second-hand and marked so; the four questions about it are closed as permanently unresolved, because this project does not consult sources it does not hold (C-31).
  • The Concordance of Failures — eighteen refusals (hyperlinked PDF) — context
    Built 19 July 2026. Six candidate refusals reached this project from commentary generated by another model; under its own discipline none of them counted, because a refusal claimed from commentary is not a refusal shown. Every one of the eighteen rows is anchored in a quotation grepped out of the 1922 text and verified back against it — all eighteen hold, checked twice by different readers. Every row's surrounding prose was corrected at least once; none passed clean, and the document says so. The one refusal type that could not be evidenced at all — the event that simply does not happen — is published as an empty row rather than removed from the chart, because an absence cannot be quoted and this dimension admits nothing it cannot quote.

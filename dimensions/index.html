The dimensions — seven kinds of thing

The project's dimensions are organised by type of operation, not by number. This embodies the project's own first finding about itself: ten of its twelve original dimensions turned out to be the same operation — extract, enumerate, tabulate — however varied their subjects looked. What a dimension does is the real difference between them.

Where the project stands: 0 built · 0 building · 16 planned · 0 dropped, of 16 dimensions. Most of the project is not built yet; this index shows that honestly.

The other entrance: every dimension has up to four output registers, and the grid can be entered by register instead — every document, every presentation, every diagram.

Type I — Inventory

The operation: extract a category, enumerate, tabulate. The output is a reference. No more than three Type-I dimensions are built (anchor A11).

  • The Object Museum — planned · the object (~30) — position hour by hour, hand to hand
  • The Women's Dublin — planned · the woman (~15) — income, mobility, interiority
  • The Governed City — planned · the instrument of rule — Castle, police, barracks, informers, language
  • The Ledger — planned · the transaction
  • The Body's Accounts — planned · the passage in or out, across nineteen hours
  • The Dead — planned · the deceased, and the mode of each return
  • The Songbook — planned · the song, the singer, and what the words do against the scene
  • The Eighteen Styles — planned · the episode's technique and what it costs the reader
  • The Apparatus — planned · Linati vs Gilbert, and the book with the grid switched off

Type II — Inversion

The operation: map the absence rather than the presence. The output is a reference of negatives.

  • The Concordance of Failures — planned · the place where Joyce refuses Homer — one column, eighteen rows

Type III — Case study

The operation: primary evidence from one reader. The output is an argument from data.

  • The Difficulty Curve — planned

Type IV — Comparative

The operation: the book against other versions of itself. The output is a parallel text.

  • Ulysses in Turkish — planned · the crux — eight to twelve common cruxes across four Turkish versions
  • Four Odysseys — planned

Type V — Model

The operation: a constructed system you operate. The output is a thing that runs.

  • The Constraint Model — planned · the constraint — a graph, a clock, a purse, and a set of obligations

Type VI — Made object

The operation: a work, not a document about a work. The output is art.

  • The Atlas of One Day — planned · the panel — eighteen, one per episode, meaning made by juxtaposition

Type VII — Instrument

The operation: a tool other people use to make data. The output is a dataset that grows.

  • The Reading Log — planned

The type is a facet, not an address: dimensions may be reclassified, and six of the Type-I dimensions may be dropped, without any link breaking. See the working document for the full reasoning behind the seven types.
